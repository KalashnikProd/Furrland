Как начать играть на сервере Furrland

1. Скачайте лаунчер Minecraft версии 1.21.4. Рекомендуется официальный или пиратский лаунчер:

Скачать лаунчер

2. Запустите Minecraft через скачанный лаунчер, выберите версию 1.21.4.

3. В Minecraft добавьте сервер с IP: play.furrland.ru

4. При первом входе зарегистрируйтесь командой:

/reg <пароль> <повтор пароля>

Пример: /reg 1234 1234

5. Чтобы установить скин, используйте команду:

/skin set <ник лицензионного игрока>

Пример: /skin set _Notch_

Посмотреть скины можно на сайте mskins.net.
