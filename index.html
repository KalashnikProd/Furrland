Добро пожаловать на сервер Furrland!

Здесь вы найдете всю необходимую информацию, чтобы начать игру и насладиться сервером.

О сервере

  • Версия клиента: 1.21.4
  • IP сервера: play.furrland.ru
  • Лаунчер (в том числе пиратский): скачать

Регистрация

При первом входе зарегистрируйтесь командой:

/reg <пароль> <повтор пароля>

Пример: /reg 1234 1234

Установка скина

Чтобы установить скин, используйте команду:

/skin set <ник лицензионного игрока>

Пример: /skin set _Notch_

Посмотреть скины можно на сайте mskins.net.
